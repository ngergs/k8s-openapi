### Optionable testing fork

Fork for testing the [`optionable`](https://github.com/ngergs/optionable) crate.

It is `k8s-openapi` v0.26.1 with the `fake::Dummy` trait derived for all types.

Will be force pushed to updated major versions when the `k8s-openapi` dependency for `optionable` gets updated. Don't use it as a dependency.
